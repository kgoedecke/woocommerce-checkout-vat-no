== Description ==

WooCommerce Checkout VAT No plugin adds a VAT number field to the checkout process of your WooCommerce store. Customers can enter their VAT number during the checkout process. The VAT no will be stored as part of the billing address and can be managed through the WooCommerce admin interface.

This plugin has the following features:
* Adds a VAT No. field to the checkout process
* VAT No. as part of the billing address
* VAT No. validation, powered by: https://github.com/dannyvankooten/vat.php
* Throws error message if an incorrect VAT number is entered

If you're facing any issues adding the VAT number field to your checkout process please contact us at [support@havealooklabs.com](mailto:support@havealooklabs.com)


== Installation ==

This section describes how to install the WooCommerce Checkout VAT No plugin and get it working .

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins interface directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> WooCommerce Checkout VAT No screen to configure the plugin


== Frequently Asked Questions ==

= How do I configured the plugin? =

There is very little configuration needed. The plugin automatically adds the VAT no field to your checkout process.

= How can I add a translation? =

Currently you have to manually add a translation file to translate the VAT no field.
